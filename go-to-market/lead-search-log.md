## Search Session Notes

- `gh` CLI searches are rate-limited. If you hit a 422/429, wait 60 seconds and retry.
- Profile lookups via `gh api users/{login}` are fast and reliable.
- Not all GitHub profiles have email — check `email`, `blog`, `bio`, and `twitter_username` fields.
- For orgs, use `gh api orgs/{org}` instead of `gh api users/{org}`.
- Email addresses on GitHub profiles are public opt-in. Using them for direct outreach is legitimate.
- Chinese QQ email addresses (like XiaoYiWeio's) may have language barrier — use short, clear messages.
